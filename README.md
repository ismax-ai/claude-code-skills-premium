# Skills Claude Code Premium

13 skills Claude Code pour les abonnés de la newsletter [Rédiger et Vendre avec l'IA](https://redigeretvendreavecia.substack.com).

## Installation

### Option 1 : une commande

Ouvre Claude Code et colle cette phrase :

```
Clone le repo https://github.com/ismax-ai/claude-code-skills-premium et copie tous les fichiers .md du dossier skills/ dans ~/.claude/commands/
```

Claude fait le reste.

### Option 2 : script

```bash
git clone https://github.com/ismax-ai/claude-code-skills-premium.git
cd claude-code-skills-premium
bash install.sh
```

### Option 3 : manuelle

Copie les fichiers `.md` du dossier `skills/` dans `~/.claude/commands/`.

## Les skills

| Skill | Ce qu'il fait |
|---|---|
| `/sentinel` | Red Team automatique. Attaque ton contenu, ton code ou tes décisions pour trouver les failles avant publication. |
| `/devils-advocate` | Challenge tes idées en prenant la position opposée. Force à considérer les angles morts. |
| `/karen` | Reality check. Vérifie ce qui est vraiment fait vs ce qui est prétendu terminé. |
| `/eagle-supervisor` | Gate final avant publication. 7 passes de vérification. |
| `/ralph` | Boucle itérative autonome. Lance une tâche et itère jusqu'à completion. |
| `/createur-de-skills` | Crée de nouveaux skills Claude Code à partir d'une description. |
| `/prompt-engineer` | Écrit et optimise des prompts structurés pour n'importe quel cas d'usage. |
| `/gouvernance-prompts` | Organise, classe et maintient ta bibliothèque de prompts. |
| `/agent-auto-ameliorant` | Agent qui s'améliore à chaque utilisation en apprenant de ses erreurs. |
| `/superpowers` | Multi-outils : debugging systématique, défense en profondeur, workflow structuré. |
| `/effort-router` | Adapte la profondeur de réflexion de Claude à la complexité de chaque tâche. |
| `/model-router` | Dirige chaque sous-tâche vers le bon modèle (Haiku/Sonnet/Opus) pour réduire les coûts. |
| `/eli5` | Passe pédagogique : détecte les termes techniques non expliqués et ajoute une définition. |

## Utilisation

Une fois installés, les skills sont disponibles dans Claude Code. Tape `/` suivi du nom du skill.

Chaque skill contient sa documentation complète en en-tête. Tu peux la lire en ouvrant le fichier `.md` ou en tapant le nom du skill dans Claude Code.

## Versions Google Doc

Si tu n'utilises pas Claude Code, chaque skill existe aussi en version Google Doc dans le dossier partagé sur Google Drive (lien dans ta newsletter de bienvenue premium).
